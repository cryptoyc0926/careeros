# CareerOS — HuggingFace Space 版

> 这是公开 Demo（BYO-Key 模式）。填入你自己的 Claude API Key 才能生成内容。

## 如何使用

1. 进入「**系统设置**」
2. 填入 `ANTHROPIC_API_KEY`（去 [Anthropic Console](https://console.anthropic.com/) 申请，有 $5 免费额度）
3. 填入个人画像（姓名/学校/专业/目标岗位）
4. 去「**主简历**」填你的简历内容
5. 去「**添加岗位 JD**」粘贴 JD 或链接
6. 去「**AI 一键生成**」或「**在线定制编辑**」出简历

## 想要自己的实例？

- **本地部署**（推荐）：[GitHub 仓库](https://github.com/careeros/careeros)
- **Fork 此 Space** → 改成你自己的 HF Space（完全免费）

## 功能

- 🎯 JD 入库（4 种方式）
- 📝 AI 按 JD 定制简历（match_score + PDF 输出）
- 📊 投递看板 + 漏斗 + 跟进提醒
- 💬 STAR 故事库 + 八股题 + 群面题
- 📧 内推/投递邮件模板一键生成
