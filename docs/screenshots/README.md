## 截图规范

- 分辨率：1600×900 或更高
- 格式：PNG（avoid JPEG 失真）
- 隐私：截图前切换到 `data/user_profile.example.yaml` 的占位画像，不要露出你的真实姓名/学校/邮箱
- 命名：全小写 + 下划线，匹配上方清单

## 建议工具

- macOS：`Cmd + Shift + 4` → 空格 → 点击窗口
- Chrome DevTools：设备模拟 1600x900 → 截屏
